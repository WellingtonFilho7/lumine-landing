# Seção 07 — Apoio recorrente

## Propósito
Para quem quer apoiar mensalmente, não apenas uma vez. Apresentar valores sugeridos sem pressão. Mostrar que constância importa mais que valor.

## Fundo
Branco (--bg-white)

## Conteúdo esperado
- Explicação breve: o projeto precisa de sustentabilidade, não só de um impulso inicial
- Valores sugeridos: R$ 50 / R$ 150 / R$ 300 por mês
- Cada valor com uma frase curta contextualizando (ex: "R$ 50/mês contribui para material de uma criança")
- A doação recorrente é via PIX agendado (não há plataforma de assinatura)
- CTA: direcionar para /doar com instruções

## Layout
- Container narrow a padrão
- Valores como badges ou chips horizontais (não cards altos)
- Texto editorial entre os valores
- Não parecer tabela de preços de SaaS

## Tom
Convidativo sem ser insistente. "Se puder, considere."

## O que NÃO fazer
- Não criar planos com nomes (Bronze, Prata, Ouro)
- Não gamificar doação
- Não mostrar "benefícios por nível"
- Não forçar recorrência como única opção
