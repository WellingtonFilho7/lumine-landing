# Seção 03 — O que é o Instituto Lumine

## Propósito
Explicar o instituto como organização. Não é uma escola tradicional, não é homeschool, não é cursinho. É uma associação sem fins lucrativos que oferece educação cristã clássica.

## Fundo
Branco (--bg-white)

## Conteúdo esperado
- Associação sem fins lucrativos (CNPJ ativo)
- Educação cristã clássica como método (breve — Trivium será para quem quer aprofundar)
- Público: crianças em idade escolar de Gurinhém e região
- Gratuito para as famílias
- Financiado por doações

## Layout
- Container padrão (72rem)
- Possível layout em 2 colunas desktop: texto à esquerda, pontos-chave à direita
- Ou editorial puro com destaques tipográficos
- Sem cards com ícones genéricos

## Tom
Explicativo, institucional. Como se estivesse respondendo "me explica o que vocês são" de forma clara e completa, sem jargão.

## O que NÃO fazer
- Não entrar em detalhes pedagógicos profundos (isso é para o FAQ ou material separado)
- Não usar termos em latim sem explicar
- Não parecer que o projeto já está funcionando
