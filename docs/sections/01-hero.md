# Seção 01 — Hero

## Propósito
Responder em 5 segundos: o que é, onde é, quando começa. Não explicar metodologia. Não emocionar. Informar com dignidade.

## Fundo
Navy escuro (--bg-navy). Texto branco. Logo centralizado ou no topo.

## Estrutura

```
[Logo do Instituto Lumine — brasão completo, tamanho médio]

Educação cristã clássica
para crianças de Gurinhém, Paraíba.

Projeto piloto com abertura prevista para março de 2026.

[Conhecer o projeto]  [Apoiar a abertura]
```

## Elementos obrigatórios
1. Logo (brasão completo, versão para fundo escuro)
2. Título principal: pode ser "Instituto Lumine" grande ou uma frase-título
3. Subtítulo: uma linha explicando o que é + onde
4. Localidade: Gurinhém, Paraíba
5. Data: março de 2026
6. CTA primário: "Conhecer o projeto" → scroll para seção 2
7. CTA secundário: "Apoiar a abertura" → link para /doar

## Layout
- Centralizado vertical e horizontalmente
- Full viewport height ou quase (min-height: 90vh)
- Não usar background image
- Espaço generoso, nada apertado
- Mobile: stack vertical dos CTAs

## O que NÃO fazer
- Não escrever parágrafos longos
- Não explicar Trivium ou metodologia
- Não usar frases como "Transformando vidas" ou "Construindo o futuro"
- Não usar countdown timer
- Não usar hero image de banco
